# has-regex-chars [![NPM version](https://badge.fury.io/js/has-regex-chars.svg)](http://badge.fury.io/js/has-regex-chars)

> Returns true if a string has regular expression special characters.

## Install with [npm](npmjs.org)

```bash
npm i has-regex-chars --save
```

## Usage

```js
var hasRegexChars = require('has-regex-chars');

hasRegexChars('$100');
//=> true

// all of the following are `true`
hasRegexChars('(');
hasRegexChars(')');
hasRegexChars('*');
hasRegexChars('+');
hasRegexChars('.');
hasRegexChars('/');
hasRegexChars('?');
hasRegexChars('[');
hasRegexChars('\\');
hasRegexChars(']');
hasRegexChars('^');
hasRegexChars('{');
hasRegexChars('|');
hasRegexChars('}');
```

## Run tests

Install dev dependencies:

```bash
npm i -d && npm test
```
